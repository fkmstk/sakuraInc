# Support

If you find a bug or need help with the Safari extension:

- Open an issue on GitHub: https://github.com/fkmstk/sakuraInc/issues
- Include the affected ASIN, device, iOS/iPadOS version, and a short reproduction note.
- If the source page layout changed, attach the product URL and observed fallback behavior.

This project does not require user accounts, subscriptions, or payment support.
